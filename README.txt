PROJECT DESCRIPTION
------------------------------------
Starcar is a web-based car rental system designed to provide an easy and secure
platform for customers to rent cars online and for administrators to manage
cars, users, and bookings.

The system supports role-based access where clients can browse cars and make
bookings, while administrators can add, edit, and manage car listings and view
user bookings.

------------------------------------
KEY FEATURES
------------------------------------
Client Features:
- Client registration and login
- Browse available cars with images
- Search and filter cars
- Select rental start and end dates
- Book cars online
- View booking history
- Secure logout with confirmation

Admin Features:
- Admin login
- Add new cars with image upload
- Edit and delete car details
- Manage car availability
- View all users
- View all bookings
- Dashboard with system statistics
